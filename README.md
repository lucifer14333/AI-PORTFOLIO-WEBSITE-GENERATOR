An AI-based platform that auto-generates professional portfolio websites using minimal user Input and Input of Resume

A sophisticated, AI-enhanced portfolio generator with dark/light mode, multiple templates, and elegant design. Create stunning professional portfolios in minutes.

**Live Demo: [https://ai-portfolio-generator.netlify.app](https://ai-portfolio-generator-ai-vengers.netlify.app/)**

---

##  Features

###  **AI-Powered Enhancements**
- **Smart Content Generation** - Creates professional portfolio content
- **Skill Categorization** - Automatically organizes skills into Technical/Soft/Tools
- **Project Enhancement** - Improves project descriptions and adds impact metrics
- **Title Suggestions** - Provides professional title recommendations

###  **Features**
- **Responsive Design** - Works perfectly on all devices
- **Drag & Drop Uploads** - Easy photo and resume uploads
- **GitHub Integration** - Fetch and add GitHub projects automatically
- **Real-time Preview** - See changes instantly
- **Export Options**:
  - Download as HTML file
  - Copy HTML to clipboard
  - One-click GitHub Pages deployment
- **Firebase Authentication** - Secure sign-in with Google, GitHub, or Email
